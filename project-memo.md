# 项目记忆：留声画 - AI照片语音解说生成器

## 基本信息
- **创建时间**：2026/04/14
- **用户身份**：大二学生，数字媒体技术专业
- **项目状态**：构思阶段

---

## 项目概述

### 痛点场景
- 手机里存了大量照片，时间久了忘了当时的故事
- 想分享给家人朋友，但解说文字太干巴巴
- 老照片背后有故事，但没人愿意花时间写

### 核心创意
**上传一张照片，AI自动分析场景并生成语音解说 + 背景音乐**

---

## 技术栈（免费工具）

| 模块 | 技术选型 |
|------|----------|
| 前端 | Vue3 + 响应式页面 |
| 后端 | Python FastAPI |
| AI图像分析 | BLIP-2 (Salesforce开源) - 图像caption生成 |
| 语音合成 | Edge-TTS (微软开源) - 免费中文语音 |
| 背景音乐 | MusicGen (Meta开源) - 根据场景生成配乐 |
| 部署 | 免费：Railway / Render |

---

## 实现流程

```
照片 → BLIP-2分析场景 → 生成文字描述
     → Edge-TTS语音合成 → MusicGen生成配乐
     → 合成带解说的短视频
```

---

## 创新点

1. **情感价值**：不只是修图，而是让回忆"活起来"
2. **自动化程度高**：上传即生成，无需用户手动编辑
3. **可扩展性强**：后续可加VR/AR展示、照片书生成
