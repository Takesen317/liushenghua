#### 问题1：TaskList.vue 删除功能显示两条消息

**问题描述**：
- 点击删除按钮后，显示"删除成功"和"删除失败"两条消息
- 任务留在列表中，刷新页面后才消失

**根本原因**：
1. `defineProps` 返回值未赋值给变量，导致 `tasks` 在 JavaScript 中未定义
2. 代码中使用了 `tasks.value.findIndex()` 和 `tasks.value.splice()`，但 `tasks` 是 prop 不是 ref

**错误代码**：
```javascript
defineProps({
  tasks: {
    type: Array,
    default: () => []
  }
})
// tasks 在这里未定义！

const deleteTask = async (task) => {
  const taskIndex = tasks.value.findIndex(t => t.id === task.id)  // 报错
  tasks.value.splice(taskIndex, 1)  // 报错
}
```

**解决方案**：
```javascript
const { tasks } = defineProps({
  tasks: {
    type: Array,
    default: () => []
  }
})

const deleteTask = async (task) => {
  const taskIndex = tasks.findIndex(t => t.id === task.id)
  tasks.splice(taskIndex, 1)
}
```

#### 问题2：share.py HTTP状态码引用错误

**问题描述**：
- 访问分享API时返回500错误
- 错误信息：`NameError: name 'status' is not defined`

**根本原因**：
- `share.py` 中使用了 `status.HTTP_404_NOT_FOUND`，但未导入 `status`
- 其他 API 文件（如 `tasks.py`）正确使用了 `from fastapi import status`

**错误代码**：
```python
from fastapi import APIRouter, Depends, HTTPException, Query
# status 未导入！

if not task:
    raise HTTPException(
        status_code=status.HTTP_404_NOT_FOUND,  # 报错
        detail="任务不存在"
    )
```

**解决方案**：
```python
if not task:
    raise HTTPException(
        status_code=404,  # 直接使用数字
        detail="任务不存在"
    )
```
