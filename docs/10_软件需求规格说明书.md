## 2. 总体描述

### 2.1 产品背景

"留声画"是一款基于人工智能技术的照片语音解说生成器，旨在解决用户照片"沉睡"、回忆难以唤起的问题。用户上传照片后，系统自动分析照片场景，生成富有情感的文字描述，并将其转化为自然语音解说，同时生成与场景匹配的背景音乐，最终合成一段可分享的短视频。

本产品面向以下主要用户群体：
- **家庭记录者**：35-55岁，已婚已育，热衷于记录孩子成长和旅行回忆
- **银发一族**：55-70岁，退休人员，珍视家庭聚会和老年生活记录
- **旅行爱好者**：22-40岁，年轻人群，习惯用照片记录旅行经历
- **学生群体**：18-25岁，大学生，记录校园生活和毕业时刻

### 2.2 用户特征

| 用户类型 | 技术水平 | 主要需求 | 使用频率 |
|----------|----------|----------|----------|
| 家庭记录者 | 中低 | 简单易用，一键生成高质量视频 | 月均2-5次 |
| 银发一族 | 低 | 操作简单，界面友好，等待时间可接受 | 月均1-3次 |
| 旅行爱好者 | 中高 | 功能丰富，支持批量处理 | 旅行期间高频 |
| 学生群体 | 高 | 免费使用，愿意尝试新功能 | 学期内定期 |

### 2.3 产品定位与竞争优势

与市场上现有产品相比，"留声画"具备以下竞争优势：

### 2.4 约束

#### 2.4.1 法规约束

- 遵守《个人信息保护法》，处理用户个人信息需获得明确同意
- 遵守《数据安全法》，对用户照片按敏感数据进行处理
- 遵守《互联网信息服务深度合成管理规定》，AI生成内容需有明确标识
- 遵守《著作权法》，用户上传照片需确认拥有版权
- MusicGen采用CC-BY-NC许可证，仅限非商业用途

#### 2.4.2 技术约束

- AI推理需本地GPU支持（NVIDIA显卡，6GB以上显存）
- Edge-TTS需联网调用微软云服务
- 音乐生成采用本地推理方式，依赖MusicGen模型

#### 2.4.3 性能约束

- 单个照片处理时间不超过60秒（使用RTX 3060显卡）
- 系统需支持并发处理多个任务
- 前端页面加载时间不超过3秒

### 2.5 假设与依赖

#### 2.5.1 假设

- 用户具备NVIDIA系列GPU（6GB以上显存）
- 用户具有稳定的互联网连接
- 用户上传的照片为JPG/PNG/WEBP格式
- 用户照片为自己拍摄或拥有版权
- MusicGen生成的音乐仅用于个人/学术目的

#### 2.5.2 依赖关系

| 依赖项 | 类型 | 说明 |
|--------|------|------|
| BLIP-2模型 | 必需 | 本地运行，用于图像分析 |
| Edge-TTS服务 | 必需 | 微软云服务，用于语音合成 |
| MusicGen模型 | 必需 | 本地运行，用于音乐生成 |
| FFmpeg工具 | 必需 | 本地运行，用于视频合成 |
| Redis服务 | 可选 | 用于Celery任务队列（无则同步处理） |

## 3. 详细需求

### 3.1 功能需求

#### 3.1.1 用户认证模块

**【功能编号】** F-AUTH-001

**【功能描述】** 系统应提供用户注册、登录和认证功能

**【功能详情】**

**1. 用户注册**
- 支持邮箱注册，需填写邮箱、密码、昵称
- 密码长度8-32位，需包含字母和数字
- 邮箱格式需符合标准规范并通过格式验证
- 昵称长度2-20个字符
- 注册成功后自动登录并生成JWT令牌

**2. 用户登录**
- 支持邮箱+密码登录
- 登录成功后返回JWT访问令牌
- 令牌有效期24小时
- 支持登录状态保持

**3. 令牌验证**
- 所有需认证的API请求需携带有效JWT令牌
- 令牌验证失败返回401状态码
- 支持令牌刷新机制

#### 3.1.2 文件管理模块

**【功能编号】** F-FILE-001

**【功能描述】** 系统应提供照片文件的上传、存储、查询和删除功能

**【功能详情】**

**1. 文件上传**
- 支持拖拽上传和点击上传两种方式
- 支持格式：JPG、JPEG、PNG、WEBP
- 单文件大小限制：20MB
- 支持上传进度实时显示
- 上传成功后返回文件ID

**2. 文件查询**
- 用户可查询自己上传的所有文件列表
- 支持分页展示（默认每页20条）
- 可查看文件的基本信息（文件名、大小、上传时间）

**3. 文件删除**
- 用户可删除自己上传的文件
- 删除文件将同时删除关联的任务
- 系统自动清理超过30天的未关联文件

#### 3.1.3 任务管理模块

**【功能编号】** F-TASK-001

**【功能描述】** 系统应提供AI语音解说生成任务的创建、查询、取消、重试和删除功能

**【功能详情】**

**1. 任务创建**
- 用户选择已上传的照片作为输入
- 支持选择解说风格：温暖（warm）、活泼（lively）、抒情（lyrical）、记录（documentary）
- 支持选择语音类型：晓晓（xiaoxiao）、云希（yunyang）、云夏（yunxia）
- 支持选择音乐风格：轻柔（gentle）、欢快（cheerful）、抒情（melancholy）、史诗（epic）
- 任务创建后返回任务ID

**2. 任务状态查询**
- 用户可实时查询任务处理状态
- 任务状态包括：pending（待处理）、processing（处理中）、completed（已完成）、failed（失败）
- 返回当前处理进度（0-100%）
- 返回当前处理步骤描述

**3. 任务取消**
- 用户可取消处于pending或processing状态的任务
- 取消后任务状态标记为failed

**4. 任务重试**
- 用户可对failed状态的任务进行重试
- 重试将重新执行全部AI处理流程

**5. 任务删除**
- 用户可删除已完成的任务
- 删除任务将同时删除关联的生成文件和分享记录

#### 3.1.4 AI处理模块

**【功能编号】** F-AI-001

**【功能描述】** 系统应实现完整的AI语音解说生成流水线

**【功能详情】**

AI处理包含以下五个步骤：

**1. 图像分析（BLIP-2）**
- 输入：用户上传的照片文件
- 处理：使用BLIP-2模型分析图像内容，生成场景描述
- 输出：图像的文字描述（AI Caption）
- 处理时间：约2-5秒

**2. 风格增强**
- 输入：AI生成的原始描述
- 处理：根据用户选择的风格（warm/lively/lyrical/documentary）调整描述
- 输出：带有风格前缀的增强描述
- 处理时间：小于1秒

**3. 语音合成（Edge-TTS）**
- 输入：增强后的描述文字
- 处理：使用Edge-TTS将文字转化为语音
- 输出：MP3格式音频文件
- 支持48kHz高清音频
- 处理时间：约3-8秒

**4. 音乐生成（MusicGen）**
- 输入：AI原始描述（用于匹配音乐风格）
- 处理：使用MusicGen生成30秒背景音乐
- 输出：WAV格式音频文件
- 处理时间：约10-20秒

**5. 视频合成（FFmpeg）**
- 输入：原始照片 + 语音MP3 + 背景音乐WAV
- 处理：使用FFmpeg将所有素材合成为短视频
- 输出：MP4格式视频文件，分辨率1080p
- 处理时间：约5-15秒

**AI Pipeline总览：**

```
照片上传 → BLIP-2图像分析 → 风格增强 → Edge-TTS语音合成 → MusicGen音乐生成 → FFmpeg视频合成 → 输出MP4
```

#### 3.1.5 分享模块

**【功能编号】** F-SHARE-001

**【功能描述】** 系统应提供视频分享功能，生成链接供他人查看

**【功能详情】**

**1. 生成分享链接**
- 用户可对已完成的任务生成分享链接
- 分享码长度为8位字母数字组合
- 支持设置链接有效期（7天/30天/永不过期）
- 同一任务可生成多个分享链接

**2. 访问分享内容**
- 无需登录即可通过分享链接访问
- 展示页面包含视频播放器和简介信息
- 访问次数自动统计

**3. 分享管理**
- 用户可查看自己生成的所有分享链接
- 用户可撤销分享链接

#### 3.1.6 视频播放与下载模块

**【功能编号】** F-PLAY-001

**【功能描述】** 系统应提供视频预览、播放和下载功能

**【功能详情】**

**1. 视频预览**
- 任务完成后自动显示视频预览界面
- 支持在线播放，无需下载即可观看
- 视频播放器支持播放/暂停控制
- 显示视频时长信息

**2. 视频下载**
- 提供视频文件下载功能
- 下载文件名为：留声画_{任务ID}_{时间戳}.mp4
- 支持浏览器原生下载

### 4.2 状态机定义

**【任务状态流转】**

```
pending（待处理）→ processing（处理中）→ completed（已完成）
         ↓                    ↓
         └───── failed（失败）←─────┘
```

**【状态说明】**

- **pending**：任务已创建，等待处理
- **processing**：AI处理进行中，可查询进度
- **completed**：处理完成，可查看结果
- **failed**：处理失败，可查看错误信息或重试
