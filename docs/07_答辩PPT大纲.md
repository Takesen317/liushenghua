### 第 3 页：项目背景

**问题痛点**：
- 智能手机拍照普及，用户手机存储大量照片
- 照片"沉睡"在相册中，难以回忆当时情景
- 传统相册仅提供静态展示，无法传达情感

**用户需求**：
- 35-60 岁：家庭照片多，希望快速生成回忆视频
- 摄影爱好者：旅行照片需要生动展示
- 18-35 岁：追求新颖有趣的社交分享方式

---

### 第 4 页：项目目标

**核心目标**：将静态照片转化为带语音解说和背景音乐的短视频

**技术路线**：
```
照片上传 → AI 图像分析 → 文字描述生成
                              ↓
视频合成 ← 背景音乐生成 ← 语音解说合成
```

**应用场景**：
- 家庭回忆录
- 旅行纪录片
- 社交媒体分享

---

### 第 5 页：技术架构

**前端**：Vue3 + Vite + Element Plus + Pinia
**后端**：Python FastAPI + SQLAlchemy
**数据库**：PostgreSQL（开发用 SQLite）
**任务队列**：Celery + Redis

**AI 服务**：
| 模块 | 技术 | 说明 |
|------|------|------|
| 图像分析 | BLIP-2 (Salesforce) | 场景理解和描述生成 |
| 语音合成 | Edge-TTS (微软) | 中文语音免费合成 |
| 音乐生成 | MusicGen (Meta) | 根据场景生成背景音乐 |
| 视频合成 | FFmpeg | 合并音视频 |

### 第 8 页：AI 技术实现

**BLIP-2 图像分析**：
- 使用预训练的 BLIP-2 模型
- 分析照片场景、物体、氛围
- 生成富有情感的描述文字

**Edge-TTS 语音合成**：
- 微软开源的中文语音服务
- 支持多种音色（晓晓、云扬等）
- 完全免费，无需 API Key

**MusicGen 音乐生成**：
- Meta 开源的音乐生成模型
- 根据文字描述生成匹配的背景音乐
- 支持多种风格（温馨、动感、抒情等）

**FFmpeg 视频合成**：
- 将照片、语音、音乐合成为 MP4
- 支持多种分辨率和编码格式
