# 留声画 - 前端设计规范

**参考风格：Rally Interactive — 大胆、现代、流畅、沉浸**

---

## 1. 设计核心理念

### 1.1 关键词
- **Bold Typography** — 超大标题，粗体字重，制造视觉冲击
- **Smooth Motion** — 所有动画丝般顺滑，300-600ms 为主
- **Dark-First** — 深色主题为主，高对比度
- **Generous Space** — 大量留白，内容呼吸感
- **Scroll-Driven** — 滚动驱动的交互动效

### 1.2 设计哲学
> "让产品看起来像是花了很多钱做的"

### 6.4 滚动动画

```javascript
// Intersection Observer for scroll animations
const observer = new IntersectionObserver(
  (entries) => {
    entries.forEach((entry) => {
      if (entry.isIntersecting) {
        entry.target.classList.add('visible')
      }
    })
  },
  { threshold: 0.1 }
)

// CSS
.scroll-reveal {
  opacity: 0;
  transform: translateY(40px);
  transition: all var(--duration-slow) var(--ease-out);
}

.scroll-reveal.visible {
  opacity: 1;
  transform: translateY(0);
}
```
